# Code of Conduct

## Our Commitment

This learning repository is an open and welcoming space. Everyone who participates — whether as a learner, contributor, or maintainer — is expected to treat others with respect and act in good faith.

## Expected Behavior

- Be respectful and constructive in all discussions and submissions.
- Help others when you can, and ask clearly when you need help.
- Give credit when referencing others' work.
- Provide feedback that is specific, honest, and kind.
- Keep discussions relevant to the course content.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language of any kind.
- Sharing others' submissions without permission.
- Submitting work that is not your own.
- Spamming issues, discussions, or pull requests.

## Reporting Issues

If you experience or witness behavior that violates this code of conduct, please open a private issue or contact the repository maintainer directly.

All reports will be handled with discretion. The maintainer reserves the right to remove content or restrict access to anyone who repeatedly violates these guidelines.

## Scope

This code of conduct applies to all interactions within this repository — issues, discussions, pull requests, and any associated spaces.
